# Jacobs Gemini AI

A simple Node.js chatbot powered by Google's Gemini API.

## Setup
1. Clone the repo  
2. Run \`npm install\`  
3. Create a .env file with your API key:  
   \`GEMINI_API_KEY=your-api-key-here\`  
4. Run \`node app.js\`  
5. Open http://localhost:3000

## Notes
- Uses \`@google/generative-ai\`
- .env is gitignored
